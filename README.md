# WeatherWise 🌤️

A weather app that shows real-time weather data for any city in the world.

## Features
- 🌡️ Current temperature, humidity, wind speed
- 🌅 Feels like temperature
- 📅 5-day weather forecast
- 🌍 Search any city worldwide
- 🎨 Beautiful dark blue UI

## Built With
- HTML, CSS, JavaScript
- OpenWeatherMap API
